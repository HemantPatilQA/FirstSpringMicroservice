To run the microservice locally follow the steps mentioned below:
1. Create a Maven project using the code base from this repository
2. From the command prompt navigate to foldr "demo" [Parent folder]
3. Run "mvn clean insatll" command
4. Once the command runs successfully and you see BILD SUCCESSFULL navigate to "target" folder
5. Run "java -jar <JAR file name>" command and it will start the server on port 8080
eg, java -jar demo-0.0.1-SNAPSHOT.jar
